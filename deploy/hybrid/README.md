# Hybrid SaaS Structure

This folder contains the control plane and data plane scaffolding for hybrid SaaS pilots.

## Layout
- `control-plane/` — UI/API hosting, tenant metadata, evidence storage.
- `data-plane/` — customer-side sidecar deployment and config.

## Entry Points
- Control plane: `deploy/hybrid/control-plane/README.md`
- Data plane: `deploy/hybrid/data-plane/README.md`
